# Password-Generator
All-in-one password generator. 1-Calculates the password's entropy. 2-Cheks in breached password databases for the password. 3-Gives multiple options on generating a password.

## Getting Started
To get all the packages.
```bash
npm install
```
To launch the app.
```bash
npm run dev
```
